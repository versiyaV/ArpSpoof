
# ArpSpoof

Данный проект был создан в ознакомительных целях.

## Установка

```bash
  git clone https://github.com/versiyaV/ArpSpoof.git
  cd ArpSpoof
  make
```
    
## Использование

```bash
sudo ./bin/main
```
